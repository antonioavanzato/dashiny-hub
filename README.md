# Дашин хаб 🎵💰

Объединенное приложение для управления финансами и учета учеников (вокал/фортепиано).

## Возможности

### 💰 Финансы
- Учет доходов и расходов
- Автоматическое распределение по категориям (40% жизнь / 20% копилка / 20% маме / 20% аренда)
- История операций с подсказками

### 🎵 Ученики
- Таблица посещаемости по дням
- Учет оплаты за занятия
- Автоматический подсчет:
  - Итого за месяц
  - Налог 4%
  - Аренда
  - Чистая прибыль
- Подсветка текущего дня

## Технологии
- HTML5 / CSS3
- Tailwind CSS
- Firebase (Auth, Firestore)
- Google Fonts (Inter)

## Для входа
Используйте свои учетные данные от Firebase.
